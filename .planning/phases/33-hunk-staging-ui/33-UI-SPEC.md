---

## Visual Hierarchy

**Focal point:** The primary visual anchor on each hunk is the **hunk toolbar row** — specifically the action buttons aligned to the right side. The hunk header text (line range info) on the left serves as secondary context. Diff lines below each toolbar are tertiary content that the user scans but does not interact with directly (in this phase).

**Reading flow:** Top-to-bottom within each file's diff. The user scans from one hunk toolbar row to the next, with diff lines flowing between them. Action buttons on the right side of each toolbar row are the primary interaction targets.

## Interaction States

### Context-Dependent Button Sets

| Diff Kind | Buttons Shown | Condition |
|-----------|---------------|-----------|
| `'unstaged'` | "Stage Hunk" + "Discard Hunk" | `fd.is_binary === false` |
| `'staged'` | "Unstage Hunk" | `fd.is_binary === false` |
| `'commit'` | (none) | Always |

### In-Flight Operation Feedback

| State | Behavior |
|-------|----------|
| Idle | All hunk buttons enabled, full opacity |
| In-flight | ALL hunk buttons in the entire file disabled (`opacity: 0.4; cursor: not-allowed`) |
| Complete | Buttons re-enable after diff re-fetch resolves |

No spinner. No loading skeleton. Reduced opacity is the sole indicator.

### Keyboard Navigation

| Key | Action | Edge Behavior |
|-----|--------|---------------|
| `]` | Scroll to next hunk toolbar row (block: 'start') | Stop at last hunk (no-op) |
| `[` | Scroll to previous hunk toolbar row (block: 'start') | Stop at first hunk (no-op) |

Guard: shortcuts are suppressed when `e.target` is `INPUT`, `TEXTAREA`, or `SELECT`.

### Hunk Navigation Highlight

| Property | Value |
|----------|-------|
| Animation name | `hunk-flash` |
| Duration | `600ms` |
| Easing | `ease-out` |
| Start color | `rgba(96, 165, 250, 0.3)` (blue tint) |
| End color | `transparent` |
| Implementation | CSS `@keyframes` animation applied via class toggle + `setTimeout` remove at 600ms |

### Discard Hunk Confirmation

| Property | Value |
|----------|-------|
| Dialog type | Tauri `ask()` (native OS dialog) |
| Title | "Discard Hunk" |
| Message | "Discard this hunk? This cannot be undone." |
| Kind | `'warning'` |
| Confirm action | Proceeds with `discard_hunk` IPC call |
| Cancel action | No-op, returns to diff view |
