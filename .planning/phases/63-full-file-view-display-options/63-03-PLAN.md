Fix three related issues in `src/components/DiffPanel.svelte`:

**Fix 1 — Async race in handleViewModeChange (lines 74-81):**
Make the function `async` and `await` the store writes BEFORE calling ondiffoptionschange. This ensures `buildDiffOptions()` in RepoView reads the updated store values:

```typescript
async function handleViewModeChange(mode: ViewMode) {
    viewMode = mode;
    await setDiffViewMode(mode);
    const shouldShowFull = mode === "full";
    await setDiffShowFullFile(shouldShowFull);
    clearSelection();
    ondiffoptionschange?.();
}
```

**Fix 2 — Async race in handleIgnoreWhitespaceChange (lines 83-87):**
Same pattern -- `await` the store write before triggering the re-fetch:

```typescript
async function handleIgnoreWhitespaceChange(value: boolean) {
    ignoreWhitespace = value;
    await setDiffIgnoreWhitespace(value);
    ondiffoptionschange?.();
}
```

**Fix 3 — Toggle button flicker on mount (lines 45-48, 60-71):**
Add a `prefsLoaded` gate that hides the entire DiffPanel content (toolbar + viewer) until the async preference load completes. This prevents the one-frame flash of default (false) values:

1. Add a new state variable: `let prefsLoaded = $state(false);`
2. In the existing $effect (line 60), set `prefsLoaded = true` inside the `.then()` callback after assigning all values
3. In the template (line 344), wrap the existing div contents in an `{#if prefsLoaded}` block. When NOT loaded, show nothing (the parent background color fills the space). This approach is preferred over a loading spinner because the LazyStore load takes <50ms — the user sees the panel appear fully formed rather than flickering.

The template change:
```svelte
<div style="height: 100%; display: flex; flex-direction: column; overflow: hidden; background: var(--color-bg);">
  {#if prefsLoaded}
    <DiffToolbar ... />
    <DiffViewer ... />
  {/if}
</div>
```

Do NOT change the existing $effect loading pattern (Promise.all) -- it is correct. Only add the `prefsLoaded` gate and `await` the store writes in the handlers.
