MergeBeginResult (serde internally-tagged enum, operation_state.rs):
```rust
#[derive(serde::Serialize)]
#[serde(tag = "kind", rename_all = "snake_case")]
pub enum MergeBeginResult {
    FastForwarded { graph: GraphResult },          // kind: "fast_forwarded" — no editor
    Conflicts { graph: GraphResult },              // kind: "conflicts"      — no editor; merge-continue UI takes over
    Ready { graph: GraphResult, message: String }, // kind: "ready"          — open editor with `message`
}
```

- `merge_continue(path: String, message: Option<String>) -> Result<(), String>`  (signature UNCHANGED; finish path now always receives Some, uses --cleanup=strip)
