    **Part C: Wire dirty detection in App.svelte**

    Add a `repo-changed` listener $effect in App.svelte that updates tab.dirty for ALL tabs (not just active):

    ```typescript
    $effect(() => {
      let unlisten: (() => void) | undefined;

      listen<string>('repo-changed', async (event) => {
        const repoPath = event.payload;
        const tab = tabs.find(t => t.repoPath === repoPath);
        if (!tab) return;
        try {
          const counts = await safeInvoke<{ staged: number; unstaged: number; conflicted: number }>('get_dirty_counts', { path: repoPath });
          tab.dirty = counts.staged + counts.unstaged > 0;
        } catch {
          // non-fatal — keep previous dirty state
        }
      }).then((fn) => { unlisten = fn; });

      return () => { unlisten?.(); };
    });
    ```

    Per D-04: dirty = staged + unstaged > 0 (conflicted files are separate). Per D-05: detection via watcher events, no polling.

    Also add initial dirty check when tabs are restored: after restoring each tab with a repoPath, call `get_dirty_counts` and set `tab.dirty`:
    ```typescript
    // Inside the restore $effect, after open_repo succeeds for a tab:
    try {
      const counts = await safeInvoke<{ staged: number; unstaged: number; conflicted: number }>('get_dirty_counts', { path: pt.repoPath });
      tab.dirty = counts.staged + counts.unstaged > 0;
    } catch {
      // non-fatal
    }
    ```
