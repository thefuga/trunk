| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-69-02 | Tampering | newer (v3) file mangled by the migration | mitigate | D-16 version-gate stays BEFORE migration; v3 → RefusedNewer untouched; regression test asserts file byte-unchanged |
