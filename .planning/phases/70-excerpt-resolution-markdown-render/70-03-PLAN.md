| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-70-03-XSS | Information Disclosure | preview rendering | mitigate | Markdown is rendered as text inside a Svelte pre element via `{markdown}` interpolation. Svelte's default text interpolation escapes HTML by default. No `@html` directive is used. No XSS surface. |
| T-70-03-RACE | Tampering | button disabled state vs IPC arrival | mitigate | Server-side D-11 gate in 70-02 returns `no_comments` error; the catch branch in onGenerateClick surfaces it via showToast. Defense in depth. |
| T-70-03-WRAP | Denial of Service | preview body line wrapping | accept | Long lines do not wrap (white-space: pre); the container scrolls horizontally. This is intentional, the user reviews exactly what the AI will see. |
| T-70-03-TOKENS | Tampering | theme custom-property usage | mitigate | All preview styling via var(--color-*); no inline literals. Verified by grep gates in acceptance criteria. |

Package legitimacy: N/A, no new dependencies.
