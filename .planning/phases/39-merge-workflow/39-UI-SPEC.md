## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (Svelte + Tauri native menus) |
| Icon library | @lucide/svelte (already in use) |

## Copywriting Contract

### Context Menu Items

| Element | Copy | Conditions |
|---------|------|------------|
| Sidebar local branch merge | `Merge {branchName} into {headBranchName}` | Hidden when `isHead === true`. Hidden when HEAD is detached (no `headBranchName`). |
| Sidebar remote branch merge | `Merge {remoteName}/{branch} into {headBranchName}` | Hidden when HEAD is detached. New context menu — remote branches currently have no right-click. |
| Graph pill merge (LocalBranch) | `Merge {pill.label} into {headBranchName}` | Hidden when `pill.isHead === true`. Hidden when HEAD is detached. |
| Graph pill merge (RemoteBranch) | `Merge {pill.label} into {headBranchName}` | Hidden when HEAD is detached. New case — RemoteBranch pills currently have no context menu. |
| Overflow ref merge (LocalBranch) | `Merge {ref.short_name} into {headBranchName}` | Hidden when `ref.is_head === true`. Hidden when HEAD is detached. |
| Overflow ref merge (RemoteBranch) | `Merge {ref.short_name} into {headBranchName}` | Hidden when HEAD is detached. |
| Commit context menu merge | `Merge {clickedBranch.short_name} into {headBranchName}` | Already exists — no change to text. |

### Menu Item Position

| Menu | Order |
|------|-------|
| Sidebar local branch | Checkout, **Merge {X} into {Y}**, Separator, Rename, Delete |
| Sidebar remote branch | **Merge {remote}/{X} into {Y}** (single item menu) |
| Graph pill (LocalBranch) | **Merge {X} into {Y}**, Separator, Rename, Separator, Delete |
| Graph pill (RemoteBranch) | **Merge {X} into {Y}** (single item menu) |
| Overflow ref (LocalBranch) | **Merge {X} into {Y}**, Separator, Rename, Separator, Delete |
| Overflow ref (RemoteBranch) | **Merge {X} into {Y}** (single item menu) |

### Feedback States

| State | Behavior | Copy |
|-------|----------|------|
| Success (fast-forward) | No toast. Graph refreshes to show updated branch pointer. | (none) |
| Success (merge commit) | No toast. Graph refreshes to show new merge commit. | (none) |
| Success (conflicts) | No toast. Operation banner appears. Conflicted files shown in staging panel. | Banner: existing `Merging {source} into {target}` from OperationBanner |
| Error (git rejection) | Red error toast with git's error message. | `{git stderr message}` as error toast — fallback: `Merge failed` |
| Detached HEAD | Merge item hidden from all menus. | (none — item not shown) |
| HEAD branch (self-merge) | Merge item hidden from all menus. | (none — item not shown) |

### Empty State

Not applicable. This phase adds menu items to existing surfaces. There is no empty state scenario.

### Error State

| Scenario | Copy |
|----------|------|
| Git rejects merge (dirty workdir conflict) | Git's stderr message shown as error toast, e.g. `error: Your local changes to the following files would be overwritten by merge` |
| Git rejects merge (already merging) | Git's stderr message shown as error toast, e.g. `error: You have not concluded your merge (MERGE_HEAD exists)` |
| Generic merge failure | Fallback: `Merge failed` |

### Destructive Actions

None in this phase. Merge is non-destructive (conflicts pause, success auto-commits). Abort already has a confirmation dialog from Phase 37.

## Interaction Contract

### Click Behavior

| Interaction | Target | Result |
|-------------|--------|--------|
| Right-click local branch (sidebar) | Branch row | Native context menu appears with Merge item |
| Right-click remote branch (sidebar) | Branch row in RemoteGroup | Native context menu appears with Merge item (new) |
| Right-click local branch pill (graph) | SVG ref pill | Native context menu appears with Merge item added to top |
| Right-click remote branch pill (graph) | SVG ref pill | Native context menu appears with Merge item (new) |
| Right-click overflow ref (graph) | Overflow popover ref | Native context menu appears with Merge item added to top |
| Click "Merge X into Y" menu item | Menu item | Menu closes. `merge_branch` IPC invoked. Graph refreshes on success. Error toast on failure. Banner appears on conflict. |

### Loading State

No loading indicator during merge execution. The native menu closes immediately on click. The merge runs in the background. The graph refreshes via `repo-changed` event on success. On conflict, the operation banner appears. On error, the error toast appears. This matches existing checkout and cherry-pick behavior.

### State Transitions

```
[User right-clicks branch] --> [Context menu with Merge item]
        |
        v (click Merge)
[Menu closes, merge_branch IPC fires]
        |
        +---> [Fast-forward success] --> repo-changed --> graph refreshes (branch pointer moves)
        |
        +---> [Merge commit success] --> repo-changed --> graph refreshes (merge commit visible)
        |
        +---> [Conflicts] --> repo-changed --> OperationBanner appears + conflicted files in StagingPanel
        |                                      (existing Phase 37-38 infrastructure)
        |
        +---> [Error] --> error toast appears with git message
```
