Key architectural decisions:
- D-01: Single ci.yml workflow with parallel jobs
- D-02: Two-gate pipeline. Gate 1: biome ci, cargo fmt --check, svelte-check (parallel). Gate 2: cargo clippy -D warnings, cargo test, vitest (parallel, needs Gate 1)
- D-03: Fast checks gate slow checks to save CI minutes
- D-08: No fail-fast -- all jobs run to completion
- D-09: Clippy with -D warnings
- D-10: All jobs on ubuntu-latest
- D-11: swatinem/rust-cache with workspaces: "src-tauri -> target"
- D-12: Bun cache via oven-sh/setup-bun@v2
- D-14: No sccache, no mold linker

Research anti-pattern: Do NOT run cargo check + cargo clippy separately. Clippy is a superset of check. Run only clippy (satisfies CI-01 requirement for both).

System dependencies needed for Gate 2 Rust jobs (clippy/test):
  libwebkit2gtk-4.1-dev, build-essential, curl, wget, file, libxdo-dev, libssl-dev, libayatana-appindicator3-dev, librsvg2-dev

Gate 1 jobs do NOT need system deps (cargo fmt is syntax-only, biome and svelte-check are frontend-only).

    **Full ci.yml content to write:**

    ```yaml
    name: CI

    on:
      push:
      pull_request:

    concurrency:
      group: ci-${{ github.ref }}
      cancel-in-progress: true

    jobs:
      # ── Gate 1: Fast checks (~10-30s each) ──

      biome:
        name: Biome
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v6
          - uses: biomejs/setup-biome@v2
          - run: biome ci .

      cargo-fmt:
        name: Rust Format
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v6
          - uses: dtolnay/rust-toolchain@stable
            with:
              components: rustfmt
          - run: cargo fmt --manifest-path src-tauri/Cargo.toml --check

      svelte-check:
        name: Svelte Check
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v6
          - uses: oven-sh/setup-bun@v2
          - run: bun install --frozen-lockfile
          - run: bun run check

      # ── Gate 2: Heavy checks (only after Gate 1 passes) ──

      cargo-clippy:
        name: Clippy
        needs: [biome, cargo-fmt, svelte-check]
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v6
          - name: Install system dependencies
            run: |
              sudo apt-get update
              sudo apt-get install -y \
                libwebkit2gtk-4.1-dev \
                build-essential \
                curl \
                wget \
                file \
                libxdo-dev \
                libssl-dev \
                libayatana-appindicator3-dev \
                librsvg2-dev
          - uses: dtolnay/rust-toolchain@stable
            with:
              components: clippy
          - uses: Swatinem/rust-cache@v2
            with:
              workspaces: "src-tauri -> target"
              save-if: ${{ github.ref == 'refs/heads/main' }}
          - run: cargo clippy --manifest-path src-tauri/Cargo.toml -- -D warnings

      cargo-test:
        name: Rust Tests
        needs: [biome, cargo-fmt, svelte-check]
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v6
          - name: Install system dependencies
            run: |
              sudo apt-get update
              sudo apt-get install -y \
                libwebkit2gtk-4.1-dev \
                build-essential \
                curl \
                wget \
                file \

          - uses: Swatinem/rust-cache@v2
            with:
              workspaces: "src-tauri -> target"
              save-if: ${{ github.ref == 'refs/heads/main' }}
          - run: cargo test --manifest-path src-tauri/Cargo.toml

      vitest:
        name: Vitest
        needs: [biome, cargo-fmt, svelte-check]
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v6
          - uses: oven-sh/setup-bun@v2
          - run: bun install --frozen-lockfile
          - run: bun run test
    ```
