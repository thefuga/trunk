    ```rust
    pub fn start_watcher<R: tauri::Runtime>(path: PathBuf, app: AppHandle<R>, state: &WatcherState) {
    ```

    1. Add `use tauri::Runtime;` import (or use fully-qualified `tauri::Runtime` in the bound)

    3. The `app` is moved into the closure, so the generic `R` must satisfy `R: Runtime + 'static` (since the closure is `'static`). Check if `Runtime` already requires `'static` -- if not, add the bound: `R: Runtime + 'static`.
    4. The `move` closure captures `app` (which is `Send + 'static` for all Runtime impls), so no additional bounds needed beyond `Runtime`.

    Updated imports at the top of watcher.rs:
    ```rust
    use tauri::{AppHandle, Emitter, Runtime};
    ```

    Updated function signature:
    ```rust

    ```

    ```rust
    fn wait_for_flag(flag: &AtomicBool, timeout: Duration) -> bool {
        let deadline = Instant::now() + timeout;
        while Instant::now() < deadline {
            if flag.load(Ordering::SeqCst) {
                return true;
            }
            std::thread::sleep(Duration::from_millis(50));
        }
        flag.load(Ordering::SeqCst)
    }
    ```
