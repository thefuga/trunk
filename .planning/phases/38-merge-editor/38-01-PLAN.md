    2. Create src-tauri/src/commands/merge_editor.rs with:
       - open_repo_from_state helper (same pattern as other command modules)
       - get_merge_sides_inner(path, file_path, state_map) -> Result<MergeSides, TrunkError>:
         - Opens repo, gets index
         - Calls index.conflict_get(file_path) to get IndexConflict with ancestor/our/their
         - For each entry (ancestor, our, their): if Some, call repo.find_blob(entry.id) then String::from_utf8_lossy(blob.content()).into_owned(). If None, return empty String.
         - Returns MergeSides { base, ours, theirs }
       - save_merge_result_inner(path, file_path, content, state_map) -> Result<(), TrunkError>:
         - Opens repo, gets workdir path
         - Writes content to repo_workdir.join(file_path) via std::fs::write
         - Gets mutable index, calls index.add_path(Path::new(file_path)), index.write()
         - This clears the conflict entry and stages the file
       - get_merge_sides Tauri command wrapper (async, spawn_blocking, same pattern as get_operation_state)
       - save_merge_result Tauri command wrapper (async, spawn_blocking, emits "repo-changed" after success, same pattern as merge_continue)
       - #[cfg(test)] mod tests with the three tests from behavior block

    3. Add `pub mod merge_editor;` to src-tauri/src/commands/mod.rs
