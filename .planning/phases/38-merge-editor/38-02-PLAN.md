    **Types:**
    ```typescript
    export interface ConflictRegion {
      type: 'context' | 'conflict';
      baseLines: string[];
      oursLines: string[];
      theirsLines: string[];
    }
    ```

    **parseConflictRegions(base: string, ours: string, theirs: string): ConflictRegion[]**
    - Split each input into lines (handle empty string => [])
    - Walk through lines using three pointers (bi, oi, ti)
    - Compare: if base[bi] === ours[oi] === theirs[ti], it is context. Group consecutive context lines into one context region.
    - If any differ, it is a conflict. Group consecutive differing lines into one conflict region. Advance each pointer independently until all three match again (or one runs out).
    - When base is empty: If ours equals theirs, return single context region. Otherwise return single conflict region with all lines.
    - Implementation approach: Use a simple sequential scan. When lines diverge, scan ahead in each to find the next sync point (next line where all three match). All lines between current position and sync point form a conflict region.

    **computeOutput(regions: ConflictRegion[], takenLines: Set<string>): string**
    - Walk regions. For context: include all baseLines (or oursLines, they are identical).
    - For conflict: iterate oursLines, include line if takenLines.has(`ours-${regionIdx}-${lineIdx}`). Then iterate theirsLines, include line if takenLines.has(`theirs-${regionIdx}-${lineIdx}`).
    - Join all collected lines with '\n'.

    **takeAllCurrent(regions: ConflictRegion[]): Set<string>**
    - Return new Set with `ours-${i}-${j}` for every conflict region i, every line j in oursLines.

    **takeAllIncoming(regions: ConflictRegion[]): Set<string>**
    - Return new Set with `theirs-${i}-${j}` for every conflict region i, every line j in theirsLines.

    **toggleHunk(side: 'ours' | 'theirs', regionIdx: number, regions: ConflictRegion[], takenLines: Set<string>): Set<string>**
    - Get the region at regionIdx. Get lines for the given side.
    - Build keys: `${side}-${regionIdx}-${lineIdx}` for each line.
    - If ALL keys are in takenLines, remove them all (untoggle). Otherwise add them all (toggle on).
    - Return new Set (immutable update).

    **toggleLine(key: string, takenLines: Set<string>): Set<string>**
    - If takenLines.has(key), return new Set without it. Otherwise return new Set with it.

    **getConflictIndices(regions: ConflictRegion[]): number[]**
    - Return array of indices where region.type === 'conflict'.
