# Quick Task 260405-j41: Fix Diff View Horizontal Scroll Summary

Moved horizontal scroll from per-line overflow-x:auto to a single container-level overflow:auto on DiffViewer so all diff lines scroll together in sync when word wrap is off.

## Changes Made

### Task 1: Move horizontal scroll from individual lines to container level (c458fd3)

**Root cause:** Each diff line div had `overflow-x: {wordWrap ? 'hidden' : 'auto'}` which created independent horizontal scrollbars on every line when word wrap was off.

**Fix:**
1. **DiffViewer.svelte** -- Changed `overflow-y: auto` to `overflow: auto` so the parent container handles both horizontal and vertical scrolling
2. **HunkView.svelte** -- Removed `overflow-x: {wordWrap ? 'hidden' : 'auto'}` from each diff line's inline style; lines with `white-space: pre` naturally extend beyond the container, and the parent provides the scrollbar
3. **FullFileView.svelte** -- Same fix as HunkView
4. **SplitView.svelte** -- Changed `.split-cell` from `overflow-x: auto` to `overflow-x: hidden`; the parent DiffViewer container handles horizontal scroll for all view modes
