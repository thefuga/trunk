<task type="auto">
  <name>Task 2: Wire overlay pipeline and render three-layer SVG in CommitGraph</name>
  <files>src/components/CommitGraph.svelte</files>
  <action>
    In `src/components/CommitGraph.svelte`:

    1. **Add imports** at the top of the script block:
       ```typescript
       import { buildGraphData } from '../lib/active-lanes.js';
       import { buildOverlayPaths } from '../lib/overlay-paths.js';
       import { getVisibleOverlayElements } from '../lib/overlay-visible.js';
       import { OVERLAY_LANE_WIDTH, OVERLAY_ROW_HEIGHT, OVERLAY_DOT_RADIUS, OVERLAY_EDGE_STROKE, OVERLAY_MERGE_STROKE } from '../lib/graph-constants.js';
       ```

    2. **Add lane color helper** (same pattern as GraphCell.svelte):
       ```typescript
       const laneColor = (idx: number) => `var(--lane-${idx % 8})`;
       ```

    3. **Add coordinate helpers** for SVG positioning:
       ```typescript
       const overlayCx = (col: number) => col * OVERLAY_LANE_WIDTH + OVERLAY_LANE_WIDTH / 2;
       const overlayCy = (row: number) => row * OVERLAY_ROW_HEIGHT + OVERLAY_ROW_HEIGHT / 2;
       ```

    4. **Add reactive pipeline** — these recompute only when `displayItems` changes (NOT on scroll):
       ```typescript
       const overlayGraphData = $derived.by(() => buildGraphData(displayItems, maxColumns));
       const overlayPaths = $derived.by(() => buildOverlayPaths(overlayGraphData));
       ```
       Place these AFTER the `displayItems` derived and BEFORE the `graphSvgData` derived.

    5. **Replace the graphOverlay snippet** — update from placeholder to full rendering. Change the snippet signature to accept `visibleStart` and `visibleEnd`:
       ```svelte
       {#snippet graphOverlay(contentHeight: number, visibleStart: number, visibleEnd: number)}
       ```

       Inside the snippet, compute visible elements (this recomputes on scroll — cheap array filtering):
       ```typescript
       const visible = getVisibleOverlayElements(overlayPaths, overlayGraphData.nodes, visibleStart, visibleEnd);
       ```
       Note: Use `$derived` pattern or call directly inside the snippet — since the snippet re-renders when args change, a direct call works fine.

       Use `OVERLAY_LANE_WIDTH` for SVG width (not old `LANE_WIDTH`):
       ```
       width={Math.max(maxColumns, 1) * OVERLAY_LANE_WIDTH}
       ```

       Render three `<g>` groups inside the `<svg>`:

       **Layer 1 — Rails (behind everything):**
       ```svelte
       <g class="overlay-rails">
         {#each visible.rails as path}
           <path d={path.d} fill="none"
             stroke={laneColor(path.colorIndex)}
             stroke-width={OVERLAY_EDGE_STROKE}
             stroke-linecap="butt"
             stroke-dasharray={path.dashed ? '3 3' : 'none'} />
         {/each}
       </g>
       ```

       **Layer 2 — Connections (middle):**
       ```svelte
       <g class="overlay-connections">
         {#each visible.connections as path}
           <path d={path.d} fill="none"
             stroke={laneColor(path.colorIndex)}
             stroke-width={OVERLAY_EDGE_STROKE}
             stroke-linecap="round"
             stroke-dasharray={path.dashed ? '3 3' : 'none'} />
         {/each}
       </g>
       ```

       **Layer 3 — Dots (on top):**
       ```svelte
       <g class="overlay-dots">
         {#each visible.dots as node}
           {#if node.isWip}
             <circle cx={overlayCx(node.x)} cy={overlayCy(node.y)} r={OVERLAY_DOT_RADIUS}
               fill="none" stroke={laneColor(node.colorIndex)}
               stroke-width={OVERLAY_EDGE_STROKE} stroke-dasharray="3 3" />
           {:else if node.isStash}
             <rect
               x={overlayCx(node.x) - OVERLAY_DOT_RADIUS}
               y={overlayCy(node.y) - OVERLAY_DOT_RADIUS}
               width={OVERLAY_DOT_RADIUS * 2}
               height={OVERLAY_DOT_RADIUS * 2}
               fill={laneColor(node.colorIndex)} />
           {:else if node.isMerge}
             <circle cx={overlayCx(node.x)} cy={overlayCy(node.y)} r={OVERLAY_DOT_RADIUS}
               fill="var(--color-bg)" stroke={laneColor(node.colorIndex)}
               stroke-width={OVERLAY_MERGE_STROKE} />
           {:else}
             <circle cx={overlayCx(node.x)} cy={overlayCy(node.y)} r={OVERLAY_DOT_RADIUS}
               fill={laneColor(node.colorIndex)} />
           {/if}
         {/each}
       </g>
       ```

    **Important notes:**
    - Do NOT remove the old `graphSvgData` derived or `GraphCell` rendering — that's Phase 24 work.
    - The overlay SVG has `pointer-events: none` and `z-index: 1` — it renders on top of the graph column but passes clicks through.
    - The `getVisibleOverlayElements()` call inside the snippet is intentional — it only filters (cheap), while the heavy `buildGraphData()` and `buildOverlayPaths()` are outside the snippet as `$derived`.
