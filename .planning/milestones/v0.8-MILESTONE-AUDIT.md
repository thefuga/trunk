## Integration Check

### Cross-Phase Wiring: 3/4 flows fully wired

#### Flow 1: Merge — COMPLETE
```
Context menu → merge_branch IPC → conflict detection → StagingPanel conflicted section
→ MergeEditor (get_merge_sides) → resolution (save_merge_result) → handleFileResolved
→ auto-open next conflict → merge_continue via bottom form
```

#### Flow 2: Rebase — COMPLETE (except Skip)
```
Context menu → rebase_branch IPC → conflict pause → StagingPanel inline rebase UI
→ conflicted files section → MergeEditor → resolution → rebase_continue via inline form
```
**Gap:** No Skip button in inline rebase UI. `rebase_skip` backend exists but is unreachable from UI during rebase operations.

#### Flow 3: Interactive Rebase — COMPLETE
```
Commit context menu → get_fork_point → get_rebase_todo → RebaseEditor
→ drag/keyboard/action editing → start_interactive_rebase (with newMessage fields)
→ [mid-rebase conflict → StagingPanel inline rebase UI → MergeEditor → resolution → Continue]
```

#### Flow 4: Operation State — COMPLETE
```
get_operation_state on every loadStatus() → OperationBanner (merge/cherry-pick/revert)
OR inline rebase UI (rebase) → action buttons → IPC commands → UI refresh
```
