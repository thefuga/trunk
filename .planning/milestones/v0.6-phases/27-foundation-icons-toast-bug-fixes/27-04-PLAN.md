FIX-01 — get_dirty_counts_inner corrected body:
```rust
fn get_dirty_counts_inner(
    path: &str,
    state_map: &std::collections::HashMap<String, std::path::PathBuf>,
) -> Result<DirtyCounts, TrunkError> {
    let repo = open_repo_from_state(path, state_map)?;
    let mut opts = StatusOptions::new();
    opts.include_untracked(true)
        .include_ignored(false)
        .recurse_untracked_dirs(true);
    let statuses = repo.statuses(Some(&mut opts)).map_err(TrunkError::from)?;
    let mut staged = 0usize;
    let mut unstaged = 0usize;
    let mut conflicted = 0usize;
    for entry in statuses.iter() {
        let s = entry.status();
        if s.intersects(
            Status::INDEX_NEW | Status::INDEX_MODIFIED | Status::INDEX_DELETED
                | Status::INDEX_RENAMED | Status::INDEX_TYPECHANGE,
        ) {
            staged += 1;
        }
        if s.intersects(
            Status::WT_NEW        // <-- THE FIX
                | Status::WT_MODIFIED
                | Status::WT_DELETED
                | Status::WT_RENAMED
                | Status::WT_TYPECHANGE,
        ) {
            unstaged += 1;
        }
        if s.intersects(Status::CONFLICTED) {
            conflicted += 1;
        }
    }
    Ok(DirtyCounts { staged, unstaged, conflicted })
}
```

FIX-02 — Handle guard pattern (suppress handle ONLY for non-message trailing handles):
For columns ref, graph: suppress when they are lastVisibleColumn
For columns author, date, sha: suppress when they are lastVisibleColumn
For message column: NEVER suppress (it uses invert=true targeting author width — it is a left-edge handle, not a trailing right-edge handle)

Example for `ref` column:
```svelte

  <div ...>
    <span>Ref</span>
    {#if 'ref' !== lastVisibleColumn}
      <div class="col-resize-handle" onmousedown={(e) => startColumnResize('ref', e)}></div>
    {/if}
  </div>
{/if}
```

Columns with their handle type:
- `ref` → normal handle, suppress if last visible
- `graph` → normal handle, suppress if last visible
- `message` → inverted handle (targets author, invert=true) → NEVER suppress
- `author` → inverted handle (targets date, invert=true) → suppress if last visible
- `date` → inverted handle (targets sha, invert=true) → suppress if last visible
- `sha` → normal handle → suppress if last visible
